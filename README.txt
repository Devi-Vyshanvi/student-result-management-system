# Student Result Management System

A backend-focused mini project built using Python and SQLite. 
The system manages student details and their subject-wise marks, 
automatically calculates total, average, and grade, and retrieves complete results by roll number.

### Features
- Add and manage student details
- Record and update marks for multiple subjects
- Generate total, average, and grade automatically
- Retrieve result reports using roll number

### Tech Stack
- Python (for logic and interface)
- SQLite (for data storage and queries)

### Skills Demonstrated
- Backend development using Python
- SQL database design and integration
- Data validation, aggregation, and report generation
